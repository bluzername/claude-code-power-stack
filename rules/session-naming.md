# Session Naming

At the start of any session where the user's first message describes a substantive task (not a quick question), suggest naming the session:

> "Want me to name this session? I'd suggest: `{project}-{type}-{descriptor}`"

Use the `/rename-session` command convention: `{project}-{type}-{descriptor}`

Do NOT suggest renaming if:
- The session already has a descriptive name
- The user is asking a quick one-off question
- The user has previously declined session naming
